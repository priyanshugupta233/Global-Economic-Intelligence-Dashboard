# 🌍 Global Economic Intelligence Dashboard

An interactive data analytics dashboard for exploring global economic trends using live **World Bank Open Data API**.

The dashboard provides GDP analysis, country comparisons, economic insights, and interactive visualizations to understand global economic performance.

---

# 📌 Project Overview

The **Global Economic Intelligence Dashboard** is a Streamlit-based analytics platform that transforms global economic data into meaningful visual insights.

Users can explore:

- 🌍 Global GDP rankings
- 📈 Historical GDP trends
- 🇮🇳 India economic analysis
- 🌎 Country comparisons
- 🤖 Automated economic insights
- 🗺️ Interactive world GDP visualization

---

# ✨ Features

## 🌍 Global Overview

- Top 10 GDP countries ranking
- GDP comparison visualization
- Historical GDP growth analysis
- Global economic snapshot


## 🇮🇳 India Economic Analysis

- India's GDP overview
- GDP per capita analysis
- Economic profile visualization


## 🌎 Country Comparison

- Compare major economies
- GDP comparison charts
- GDP per capita comparison


## 🤖 Economic Intelligence

- Automated economic observations
- Largest economy identification
- GDP per capita insights


## 🗺️ World GDP Map

- Interactive world map
- Country-wise GDP visualization
- Global economic distribution
